# 🎬 Video Subtitle Grabber

> Расширение для Chrome/Edge, которое показывает медиа, встроенное в открытую веб-страницу, и помогает сохранить **дорожки субтитров/титров, уже загруженные вашим браузером** — для личного использования (доступность, офлайн-чтение, перевод, изучение языков).

## Возможности

- **Автообнаружение** видео Vimeo, YouTube, нативных `<video>` и HLS-потоков на любой странице
- **Скачивание субтитров** (.vtt / .srt) в один клик
- **Понятное имя файла** — `Название видео [ID].язык.vtt`
- **Превью** с длительностью (16:9)
- **Кликабельный бейдж платформы** — открывает видео напрямую
- **Скачать все** — сразу все дорожки субтитров на странице
- Читает текстовые дорожки, отдаваемые встроенным плеером (в т.ч. встраивания с токеном доступа, который браузер уже отправляет)
- **Режим «Кратко / Полный»** — спрятать превью для плотного списка
- **Русский и английский** интерфейс с переключением в один клик (в панели и в попапе)
- Разделы **«О плагине» / «Что нового»** и встроенная **проверка обновлений** через GitHub
- **Полная изоляция UI** через iframe в origin расширения — CSS *и* CSP страницы не влияют на панель

## Установка (режим разработчика)

1. Скачайте или клонируйте этот репозиторий
2. Откройте `chrome://extensions/` или `edge://extensions/`
3. Включите **Режим разработчика** (переключатель справа вверху)
4. Нажмите **Загрузить распакованное расширение** → выберите эту папку
5. Закрепите иконку расширения на панели инструментов

## Использование

1. Откройте любую страницу со встроенным видео
2. Нажмите иконку **🎬** на панели инструментов (в попапе можно переключить язык)
3. Откроется панель — видео определяются автоматически
4. Нажмите **⬇ Язык**, чтобы скачать субтитры
5. Или **⬇ Все (N)** — чтобы скачать все дорожки разом

> **Про HLS / Circle.so:** дорожки субтитров, объявленные в master-плейлисте m3u8, находятся через Performance API, который видит только уже запрошенные страницей потоки — **сначала запустите видео (Play)**, затем откройте панель.

## Правовая информация и назначение

Это универсальная утилита. Она показывает медиа, встроенное в открытую страницу, и позволяет сохранить **текстовые дорожки субтитров/титров, которые браузер уже загрузил** в ходе обычного воспроизведения. Она **не** скачивает само видео и **не** обходит технические средства защиты (DRM) или контроль доступа.

Предназначена для **личного законного использования**: доступность (например, для слабослышащих), офлайн-чтение, перевод и изучение языков, конспектирование.
